Buxfer lets you customize (almost) everything

Instead of locking you into pre-made rules and categories, Buxfer lets you set up and customize rules that fit your needs perfectly. For example, set up a rule to re-name a transaction if it contains a word or phrase in the name, then tag it with three categories, then apply to future and/or all transactions.

Create an unlimited number of rules, for any expense, income and transaction you can think of.
